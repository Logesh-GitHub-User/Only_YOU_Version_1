## Description

Only YOU is a simple personal diary application that allows users to write, store, encrypt, and read diary entries securely using a Java program. The entries are stored locally in an encrypted text file, ensuring privacy and security.

---

## Features

1. You can write your diary entry and it will be stored in a text file in your system.
2. The file will be **encrypted** before storing locally.
3. You can read the diary entry **only through the program**.
4. Even if someone opens the file directly, the content will be **unreadable** because it is encrypted.
